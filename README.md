# 🎋 おみくじBot

[mixi2-js](https://github.com/otoneko1102/mixi2-js) を使ったシンプルなおみくじBotのサンプルです。

Botにメンションすると、おみくじの結果を返信します。

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env.example` をコピーして `.env` を作成し、認証情報を設定してください。

```bash
cp .env.example .env
```

### 3. 起動

```bash
npm start
```
